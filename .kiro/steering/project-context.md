# Contexto del Proyecto

## Descripción

Este es un **workshop de DevOps** diseñado para aprender a usar Kiro como herramienta de desarrollo asistida por IA. El objetivo es explorar cómo Kiro puede ayudar en tareas de CI/CD, automatización y buenas prácticas de ingeniería.

## Stack Tecnológico

- **Runtime:** Node.js 20
- **Framework:** Express
- **Testing:** Jest
- **Lenguaje:** JavaScript/TypeScript

## Infraestructura y Deploy

- **Cloud provider:** AWS
- **Deploy targets:**
  - **AWS Lambda** - Para funciones serverless y APIs ligeras.
  - **AWS ECS** - Para servicios containerizados con mayor control.
- Elegir el target según la complejidad del servicio y los requisitos de escalabilidad.

## Preferencias de Comunicación

- Las respuestas, comentarios en código y documentación deben estar en **español**.
- Los nombres de variables, funciones y archivos de código permanecen en inglés (convención estándar).
- Los mensajes de commit y descripciones de PR pueden ser en español.
